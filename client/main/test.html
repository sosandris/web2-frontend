  • dialog: must be same as id on dialog

<tilos-dialog-handler dialog="#dialog-mikulas">
  <button type="button" class="btn btn-danger">Tilos Mikulás</button>
</tilos-dialog-handler>
  • header: Title of the popup
  • default-open: The dialog show if page loader, without any user action
  • id: Handler use for matching dialog, also key for store if user check default open false

<tilos-dialog header="Jön a Tilos Mikulás" default-open id="dialog-mikulas">
